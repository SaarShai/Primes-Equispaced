What you're seeing

Each dot is a fraction a/b placed at angle 2π·(a/b) on the circle. The red dots are near the top (cos > 0, fractions near 0 or 1). The teal dots are near the bottom (cos < 0, fractions near 1/2).

When M(N) > 0: more red than teal — the "even personality" numbers are winning, creating a top-heavy bias. A prime arriving with its uniform grid can heal this.

When M(N) < 0: more teal than red — bottom-heavy bias. A prime's uniform grid can't help here.

When N is prime: it fills ALL N−1 grid positions (100% fill). When N is composite: it only fills φ(N) positions, leaving holes.

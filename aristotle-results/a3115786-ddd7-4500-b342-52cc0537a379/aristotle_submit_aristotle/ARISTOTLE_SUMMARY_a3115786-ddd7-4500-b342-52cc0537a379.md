# Summary of changes
## Summary

The original `ratio_test` theorem was **unprovable as stated**. I identified the issue, provided a counterexample, and proved the corrected version.

### Problem with the original statement
The original hypothesis `newDispSquaredSum p + shiftSquaredSum p ≥ dilution p` (i.e., D + C ≥ dilution) is insufficient because it omits:
1. The **cross term** B = `crossTerm p`
2. The **boundary correction** of -1 (arising from the special behavior of f = 1 in the displacement-shift identity)

**Counterexample:** For p = 7, the hypothesis holds (D + C ≥ dilution) but `deltaWobble 7 > 0`, so the conclusion fails. This is proved as `ratio_test_counterexample` using `native_decide`.

### Corrected theorem
The sufficient condition is `D + B + C - 1 > dilution`, i.e.:
```lean
theorem ratio_test (p : ℕ) (hp : Nat.Prime p) (hp5 : 5 ≤ p)
    (h : newDispSquaredSum p + crossTerm p + shiftSquaredSum p - 1 > dilution p) :
    deltaWobble p < 0
```

### What was proved (all sorry-free, verified with `#print axioms`)

1. **`four_term_decomposition`**: The key identity `WN(p) = WN(p-1) + B + C - 1 + D`, proved by:
   - Splitting `fareySet(p)` into old and new fractions
   - Using `displacement_shift` for f ≠ 1 and the boundary correction for f = 1
   - Applying `quadratic_expansion_sum`

2. **`sign_condition`**: If `D + B + C - 1 > dilution` then `deltaWobble p < 0`, proved algebraically from the four-term decomposition by clearing denominators.

3. **`ratio_test`**: The corrected theorem, which follows directly from `sign_condition`.

4. **Helper lemmas**: `fareyCount_pos`, `displacement_one_eq_zero`, `shiftFun_one`, etc.

5. **`ratio_test_counterexample`**: Formal proof that the original hypothesis is false for p = 7.

The original statement is preserved as a comment with an explanation of why it's incorrect.

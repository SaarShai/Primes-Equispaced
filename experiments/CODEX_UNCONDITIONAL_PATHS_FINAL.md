# Codex: Unconditional Paths — Final Assessment
# 2026-04-10

## Path A — RIGHTMOST ZERO: BLOCKED
- Global rightmost zero may not exist (supremum, not maximum)
- Even with a maximizing zero, phase cancellation is the real issue
- If β* = 1/2 → back to RH regime

## Path B — DENSITY ESTIMATES: NEEDS_WORK (most promising)
- Can plausibly support "almost all zeros" or density-one statement
- Cannot give pointwise F(γ_k)/F_avg → ∞ by itself
- Next step: combine density estimates with discrete mean-value or large-values theorem

## Path C — AVERAGE OVER ZEROS: BLOCKED
- Gonek MVT gives fair share, not excess energy
- Zeros get exactly what the continuous average predicts
- Would need higher-moment or large-deviation theorem for our coefficients

## Other — EXPLICIT FORMULA/RESONATOR HYBRID: SPECULATIVE
- Only genuinely new mechanism possible
- Need: nonneg kernel at zero ordinates + prime side dominates unconditionally
- No existing theorem does this

## Ranking (by promise)
1. Path B (density-one theorem — weaker but achievable)
2. Resonator hybrid (novel but speculative)
3. Path C (fair-share blocks it)
4. Path A (weakest — extremal zero premise unresolved)

## Bottom Line
A fully unconditional POINTWISE result (F(γ_k)/F_avg → ∞ for each k) is probably 
out of reach without genuinely new tools. But a DENSITY-ONE result 
("for all but o(N(T)) zeros, F(γ_k) is larger than average") might be achievable 
via Path B + large-values framework.

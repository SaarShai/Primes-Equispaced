**arXiv Submission Checklist for Math Paper (Primary: math.NT, Secondary: math.CO)**  
**Check each item as "Yes" or "No" before submission.**

| **Item** | **Yes / No** | **Notes** |
|----------|--------------|-----------|
| **1. Abstract under 300 words** | ☐ | Ensure concise summary of research, context, and results. |
| **2. All references complete** | ☐ | Verify all citations (bibliography entries) are properly formatted and match the paper. |
| **3. No compilation errors** | ☐ | Compile the LaTeX document to confirm no syntax or formatting issues. |
| **4. Endorsement code ready** | ☐ | If required (e.g., for certain categories), ensure the endorsement code is correctly formatted and valid. |
| **5. Supplementary materials linked** | ☐ | Include links to additional files (e.g., data, code, proofs) in the "Supplementary Material" section. |
| **6. AI disclosure present** | ☐ | If AI tools were used in research, include a statement in the "AI Disclosure" section (e.g., "This work was supported by AI tools..."). |
| **7. Author list correct (single author)** | ☐ | Confirm the author list includes only the single author and matches the paper’s content. |
| **8. MSC codes correct** | ☐ | Primary code: **math.NT** (e.g., 11-XX), Secondary code: **math.CO** (e.g., 05-XX). Verify codes match the paper’s focus. |

---

**Additional Tips:**  
- **arXiv categories:** Ensure the primary category is **math.NT** and the secondary is **math.CO**.  
- **File format:** Submit as a single `.tex` file (or `.pdf` if required) with all dependencies (e.g., fonts, packages).  
- **License:** Choose an open-access license (e.g., CC BY 4.0).  
- **Title:** Ensure the title is clear, concise, and includes keywords (e.g., "Number Theory and Combinatorics").  

**Final Check:** All items marked "Yes" before submission.

## Method

The **twisted spectroscope** is defined as:

$$F_\chi(\gamma) = \left|\sum_{p \le N} \chi(p)\, p^{-1/2 - i\gamma}\right|^2$$

A Siegel zero beta (real, close to 1) of L(s,chi) would create an anomalous peak near gamma = 0 in F_chi(gamma), because the spectroscope resonates at the imaginary parts of L-function zeros, and a real zero has imaginary part 0.

We scan gamma in [0.01, 5] with 5000 points for each primitive character, looking for statistically significant peaks (z-score > 2) in the region gamma < 2.

## Null Control

The trivial character (Riemann zeta) serves as null control. Since zeta(s) has no real zeros near s = 1, we expect no anomalous peaks at low gamma.

**Result:** No significant peaks at gamma < 2. Control passes.

### Overall

With N = 1,000,000 primes, the Farey spectroscope can rule out Siegel zeros with |1-beta| > **0.01** for all primitive characters mod q with q in {3,4,5,7,8,11,12,13}.

This is a **practical, not rigorous** bound. It means: if a Siegel zero existed with |1-beta| > 0.01, the spectroscope would have detected an anomalous peak with z-score > 5 -- and no such peak was observed.

### Caveats

1. This is computational evidence, **not a proof**.
2. Sensitivity depends on N (more primes = better detection).
3. The test only covers small moduli. A Siegel zero could theoretically exist for large q.
4. The injection model (additive residue) is approximate.
5. Real Siegel zeros, if they exist, might have more subtle spectral signatures.

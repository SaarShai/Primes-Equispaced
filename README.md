# The Geometric Signature of Primes in Farey Sequences

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**The nontrivial zeros of the Riemann zeta function govern the step-by-step regularity of rational numbers.**

We introduce the **per-step Farey discrepancy** ΔW(p) — a novel measure of how prime numbers affect the uniformity of rational number distributions — and discover a direct phase-lock to the leading zeta zero γ₁ = 14.135.

## Key Discoveries

### 1. The Farey Spectroscope
A spectral function F(γ) = |Σ R(p)·p^{-1/2-iγ}|² whose peaks detect zeta zero locations from Farey sequence data alone — a different computational path to the zeros. The first zero is recovered at 14.05 (0.6% error). Extends to Dirichlet L-functions via character twisting.

### 2. Damage/Response Mechanism
When a prime enters the Farey sequence, it *damages* regularity, but existing fractions *overcompensate*. The fraction 1/p alone accounts for 65% of the damage, with the exact formula D(1/p) = 1 - |F_{p-1}|/p ≈ -3p/π².

### 3. Chebyshev Bias in Farey Discrepancy
The sign of ΔW(p) is phase-locked to cos(γ₁·log(p) + φ) with R = 0.77 across 4,617 qualifying primes ≤ 100,000.

### 4. Density Threshold for Healing
Composites with φ(n)/n < 1/3 always improve regularity (ΔW > 0). The boundary between damage and healing is controlled by the Euler product ratio.

### 5. Formal Verification
30 Lean 4 files with 260+ verified results, proofs generated by [Aristotle](https://aristotle.harmonic.fun).

## Tools for Researchers

**Farey Spectroscope** — detect zeta zeros from Farey data:
```python
python3 experiments/farey_spectroscope.py   # generates spectroscope figure
python3 experiments/unified_depth_tests.py  # runs detection + sign prediction
```

**R(p) Computation** — extend the correlation ratio dataset:
```bash
gcc -O2 -o R_bound experiments/R_bound_1M.c -lm
./R_bound 200000  # compute R₂(p) for qualifying primes up to 200K
```

**Lean 4 Proofs** — verify all formal results:
```bash
cd RequestProject && lake build
```
Requires Lean 4 v4.28.0, Mathlib v4.28.0.
